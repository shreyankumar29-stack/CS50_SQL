# 📚 About the Course

CS50's Introduction to Databases with SQL teaches the fundamentals of relational databases and SQL, covering everything from basic queries to database design and optimization.

Throughout this course, I learn how to:

- Query relational databases
- Filter and sort data
- Design database schemas
- Work with relationships
- Use aggregate functions
- Optimize SQL queries
- Build real-world database projects
