# CS50 SQL – Week 0: Querying

Welcome to **Week 0** of **CS50's Introduction to Databases with SQL**.

This folder contains my personal notes, practice queries, and code written while learning the fundamentals of SQL and relational databases through Harvard University's CS50 SQL course.
